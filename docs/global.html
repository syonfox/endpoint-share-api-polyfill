sharePolyfill :SharePolyfill

This will setup the polyfill with default options. So it will only use the polyfill if navigator.share does not exist You can still use sharePolyfill.share(); or initalize with sharePolyfill = new SharePolyfill({forcePolyfill:true}); equivalently navigator.share = sharePolyfill.share; happy hacking
